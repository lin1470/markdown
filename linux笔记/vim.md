| 快捷键              | 说明                     |
| ------------------- | ------------------------ |
| `vim -u NONE -N`    | 开启vim时不加载vimrc文件 |
| `<leader>e`         | 快速编辑vimrc文件        |
| `<leader>s`         | 重新加载vimrc文件        |
| `<leader>h`         | 打开vimplus帮助文档      |
| vim - o 文件1,文件2 | 横向打开多个文件         |
| vim -O 文件1,文件2  | 纵向打开多个文件         |
| :vs file            | 横向打开多个文件         |
| :sp file            | 纵向打开多个文件         |
